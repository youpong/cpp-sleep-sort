# README.md

The C++20 implementation of Sleep Sort.

## Prerequisites

The following versions or newer required.

* CMake 3.22
* GNU Make-4.3
* g++-11.3.0 or clang++-15.0.6
* clang-format-15.0.6

## How to use

```
$ mkdir build
$ cd build
$ cmake ..
$ cmake --build .
```
